Training Kit

Open source GitHub courseware

Present your way to better version control

With our open source Git & GitHub teaching materials, you can teach yourself or an audience, right from the convenience of your web browser

We've created a combination of a slide deck, lecture, and labs for each class we offer. Just launch any class and click the right-side TOC navigation to jump to a section.

Foundations

Gain core GitHub skills for day-to-day use of the GitHub platform, Git clients, and standard workflows.

Intermediate

Explore fundamental Git command line skills and problem solutions to creating history, effectively branching, and reverting when necessary.

Advanced

Gain a deeper understanding of Git's command line and its interaction with the GitHub platform for advanced problem solving techniques and efficiencies in using GitHub.

Cheat Sheets

Looking for a quick reference sheet of Git commands? We've got just the document for you. It's available for PDF-format download in the following languages:

English Spanish French Japanese

Community contributions

Would you like to update or clarify content in the Kit? Just fork the repository, make your changes, and submit a Pull Request.
